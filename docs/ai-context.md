# Contexto del proyecto

Aplicación móvil con Ionic Angular.

## Arquitectura
- Usar Angular standalone
- Separar lógica en services
- Componentes reutilizables
- Manejo básico de estado en memoria

## Objetivo
Crear una app móvil con:
- Pantalla principal (dashboard)
- Navegación con tabs
- Listado de datos (mock/local)
- Detalle de elementos

## Reglas
- Código limpio y tipado
- No usar backend
- No autenticación (sin login, sin JWT)
- Usar environment.ts solo si es necesario
